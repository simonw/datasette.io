LAION-Aesthetics v2 6+

{{ render_markdown(""" The Stable Diffusion text-to-image model was trained primarily using [LAION-5B](https://laion.ai/blog/laion-5b/) and [LAION-Aesthetics](https://laion.ai/blog/laion-aesthetics/), enormous datasets of images scraped from the web. [laion-aesthetic.datasette.io](https://laion-aesthetic.datasette.io/laion-aesthetic-6pls/images) presents a subset of 12 million images from LAION-Aesthetics, filtered to the images with an aesthetic score of 6 or higher. The goal is to help people understand the kind of images that are used to train these image generation models. [Exploring the training data behind Stable Diffusion](https://simonwillison.net/2022/Sep/5/laion-aesthetics-weeknotes/) explains how the Datasette instance was constructed in detail, and [Exploring 12 Million of the 2.3 Billion Images Used to Train Stable Diffusionâ€™s Image Generator](https://waxy.org/2022/08/exploring-12-million-of-the-images-used-to-train-stable-diffusions-image-generator/) describes the results of the exploration. """) }}

California Protected Areas

{{ render_markdown(""" CPAD, the [California Protected Areas Database](https://www.calands.org/cpad/), is a remarkable GIS dataset maintained by [GreenInfo Network](https://www.greeninfo.org/), an Oakland non-profit. This example uses the [shapefile-to-sqlite](https://datasette.io/tools/shapefile-to-sqlite) tool to import Shapefiles from that project into a [SpatiaLite](https://www.gaia-gis.it/fossil/libspatialite/index) database, then demonstrates a number of different Datasette geospatial plugins, including: - [datasette-leaflet-geojson](https://datasette.io/plugins/datasette-leaflet-geojson) to render the data on a map - [datasette-leaflet-freedraw](https://datasette.io/plugins/datasette-leaflet-freedraw) to allow users to draw shapes on the map and query the database for matching records For more on this project, see [California Protected Areas in Datasette](https://simonwillison.net/2020/Aug/28/weeknotes-cpad/) and [Drawing shapes on a map to query a SpatiaLite database](https://simonwillison.net/2021/Jan/24/drawing-shapes-spatialite/). [Building a location to time zone API with SpatiaLite](https://datasette.io/tutorials/spatialite) is a tutorial showing how to use SpatiaLite with Datasette. """) }}

San Francisco Microscopical Society Archive

{{ render_markdown(""" The [San Francisco Microscopical Society](https://sfmicrosociety.org/) have archives dating back to 1870, and recently started digitizing them as scanned images in PDF files. [s3-ocr](https://datasette.io/tools/s3-ocr) is a tool for bulk OCRing PDF files in an S3 bucket and extracting out the resulting text into a searchable SQLite database. Their [archive site](https://archive.sfmicrosociety.org/) uses a templated Datasette instance to allow people to search and browse through their archives. [Building a searchable archive for the San Francisco Microscopical Society](https://simonwillison.net/2022/Aug/25/sfms-archive/) describes in detail how this project was built. """) }}

Global Power Plants

{{ render_markdown(""" The World Resources Institute publishes a [Global Power Plants Database](https://www.wri.org/research/global-database-power-plants) with information about over 30,000 power plants around the world. [This Datasette instance](https://global-power-plants.datasettes.com/global-power-plants/global-power-plants) presents those power plants on a map, using the [datasette-cluster-map](https://datasette.io/plugins/datasette-cluster-map) plugin to support zooming in to see individual plants. """) }}
